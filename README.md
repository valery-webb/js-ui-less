js-ui-less 
==========

1. Develop:
  * Clone this repo
  * Install nodejs
  * Install grunt
  * Make build and watch by command 'grunt'

2. Use:
  * Just clone this repo and run index.html

<br />
<br />
compatibility: IE8+
<br />
custom grid system: utils.less

polyfills: html5shiv, respondjs
<br />
jquery: 1.8.0
